## 🎯 项目简介

**LogicScale** 是一款创新的购物决策工具，将复杂的产品对比转化为直观的 3D 物理仿真天平实验。通过将产品的优点转化为金属砝码，缺点转化为悬浮气球，以物理倾斜度直观呈现产品优劣。

### 核心价值
- **决策物理学**：将心理偏好量化为物理受力分析系统
- **智能偏差过滤**：识别并修正9大认知偏差
- **极速决策路径**：为不同用户画像提供个性化建议
- **致命一击分析**：识别能瞬间反转决策的关键因素

## 🚀 功能特性

### 1. 语义门控与路径识别
- **极速预判模式**：快速对比两款产品
- **标准/偏差模式**：深度分析多维度需求
- **智能Fallback**：当输入信息不足时自动追问

### 2. 专家级9大反模式过滤
- 现状偏差 (Status Quo)
- 投射偏差 (Projection)
- 确认偏差 (Confirmation)
- 最近效应 (Recency)
- 参数空心化 (Hollow)
- 低价锚定 (Anchor)
- 吸铁石效应 (Sunk Cost)
- 可用性启发 (Availability)
- 框架效应 (Framing)

### 3. 决策矩阵计算框架
- 品类默认权重分配（手机、电脑、相机、耳机等）
- 科学的计分规则（损失厌恶校准）
- 胜率计算与可视化

### 4. 胜负判定与"致命一击"协议
- 极端探测测试
- 压力测试
- 关键因素识别

### 5. 模块化输出规范
- 3D仿真主视觉
- 砝码与气球矩阵
- 决策物理学报告
- 极速决策路径
- 致命一击提示
- 追问与重算协议
- 自适应追问模板

## 🎨 视觉规范

## 📦 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/yourusername/logic-scale.git

# 进入项目目录
cd logic-scale

# 安装依赖
npm install
```

### 使用

1. **启动服务**：`npm start`
2. **访问界面**：打开浏览器访问 `http://localhost:3000`
3. **输入问题**：例如 "iPhone 15 和 16 哪个好？"
4. **查看结果**：获得详细的决策分析报告

## 🔧 技术栈

- **前端**：React + TypeScript + Three.js
- **后端**：Node.js + Express
- **AI**：自定义决策引擎
- **UI**：Tailwind CSS + Framer Motion
